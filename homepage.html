Grundlagen der Barrierefreiheit im Web

Barrierefreiheit (Accessibility) bedeutet, dass Webseiten so gestaltet werden, dass sie von allen Menschen genutzt werden können - unabhängig von körperlichen Fähigkeiten oder technischen Möglichkeiten. Das Video zeigt die wichtigsten Prinzipien für eine zugängliche Website-Gestaltung.
